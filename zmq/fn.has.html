Function zmq::has[−][src]

pub fn has(capability: &str) -> Option<bool>

Return true if the used 0MQ library has the given capability.

The return value is always the Some variant; it used to return None for older, now unsupported versions of 0MQ that didn’t have the wrapped zmq_has function. Thus, for code that requires zmq version 0.9.0 or newer, you can safely call unwrap on the return value.

For a list of capabilities, please consult the zmq_has manual page.

Compatibility

In the zmq 0.10.0, this function will simply return bool.
